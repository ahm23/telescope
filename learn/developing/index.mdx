# Learning to Develop with Telescope

Welcome to your journey of contributing to Telescope! This guide will walk you through everything you need to know to become a successful Telescope developer.

## Your Learning Path

Start with these fundamental guides:

- [Setting Up Your Environment](/learn/developing/setup) - Get your development environment ready
- [Running Tests](/learn/developing/tests) - Learn how to test your code
- [Using Starship for Testing](/learn/developing/starship-tests) - Master end-to-end testing
- [Understanding Packages](/learn/developing/packages) - Explore how Telescope is structured
- [Working with ASTs](/learn/developing/working-with-asts) - Learn to manipulate Abstract Syntax Trees
- [Using Common Helpers](/learn/developing/helpers) - Leverage existing utilities
- [Creating New Generators](/learn/developing/creating-new-generators) - Build your own code generators
- [Migrating to v1.0](/learn/developing/migration) - Update your code to the latest version
- [Using Tree-Shakable Hooks](/learn/developing/tree-shakable-hooks) - Implement efficient hooks

## Getting Started

We recommend starting with the [Setup Guide](/learn/developing/setup) to prepare your environment, then moving on to [Understanding Packages](/learn/developing/packages) to get familiar with the codebase structure.

## Need Help?

If you get stuck at any point:
- Check related reference documentation in the [docs section](/telescope/developing)
- Join our community channels for support
- Open an issue on GitHub

Happy coding! We're excited to see what you'll build with Telescope.
